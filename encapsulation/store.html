# 简述

  • 使用 Vuex 做状态管理的数据，仅包含用户信息，机构信息等内容，此处不做赘述。
  • 云柜台交易页面数据主要通过 session 和报文来获取。

# session 和报文

  • 在 app-loader.js 中通过 SessionHelper 获取到的信息，按 Client.sdb 的需要赋值到了 Ameba.session.Client 上。
  • 交易中用到的数据，主要来自通过监听 openPage 事件收到的报文。

# 云柜台导航页面 PageContainerNavigator.vue

  • PageContainerNavigator 作为云柜台交易页面的父组件，引入 Ameba 将获取到的 session 赋值到 this.session 上，再通过 :session="session" 绑定到动态子组件 <component/> 的 props 上实现父子组件传值 。

  • 触发类型为 openPage 的事件消息即交易所用报文，将报文 pageInfo 赋值到 this.context 上，再通过 :context="context" 绑定到动态子组件 <component/> 的 props 上实现父子组件传值 。

<template>
  <div class="page-container-navigator" id="remotepagecontainernavigator">
    <transition name='page' mode="out-in" @after-enter='onNavigated'>
        <component ref="currentPage" class="page trade-content-font" :key="indexDate" :is="currentPage" :BUSISTEP="BUSISTEP" :session="session" :context="context" @openPage="openPage" @exit="exitTrade" @abort="abortTrade"></component>
    </transition>
  </div>
</template>

<script>
import { Ameba } from "awp-lib-ameba-js";
export default {
  name: "page-container-navigator",
  data() {
    return {
      session: undefined,
    };
  },
  methods: {
    async onInitialize() {
      // 获取session
      let result = Ameba.session;
      this.session = result;
      //注册remote事件
      RemoteCommandManager.addHandler(
        RemoteCommandTypes.openPage,
        this.onOpenPage.bind(this)
      );      
    },
    async onOpenPage(pageInfo) {
      this.context = pageInfo;
    },
};
</script>

# 交易进度页面 IdlePage.vue

  • IdlePage 是交易等待过程中的交易进度提示及广告页面。
  • 通过 this.context 获取报文中的信息。
<template>
  <page-group>
    <page-header :hasUnderline='false'></page-header>
    <page-content class="paga-content">
      <aui-box class="info-box" direction="column" justify-content="center">
        <aui-box class="tips-div" direction="column" justify-content="flex-start"align-items="flex-start">
          <div class="ageng-sytle">         
            <span class="busi-name">{{agentId}}</span>号云柜员<span class="busi-name">{{agentName}</span>正在为您服务，请耐心等待
          </div>
        </aui-box>
        <aui-box class="info-box-div" direction="column" justify-content="space-around">
          <aui-box class="top-div">
            <aui-box class="img-box">
              <img class="adver-img" src="static/resource/trade/ayo/广告图.png">
            </aui-box>
          </aui-box>
          <aui-box class="progress-box">
            <aui-progress :percentage="compRate" :text-inside="true" :stroke-width="27"strokeColor="#ff4949" :striped="true">当前业务已完成 {{compRate}}%</aui-progress>
          </aui-box>
          <aui-box class="tips-div process-tips" direction="column" justify-content="flex-start"align-items="flex-start">
            <div>正在为您办理
              <span class="busi-name">{{flowName}}业务</span>,当前正在处理
              <span class="node-name">{{nodeName}}</span>请耐心等待，谢谢您的合作
            </div>
          </aui-box>
        </aui-box>
      </aui-box>
    </page-content>
    <page-footer></page-footer>
  </page-group>
</template>

<script>
import BasicPage from "@main-stm/src/sdk/mixins/basic-page-noflow";
export default {
  name: "idle-page",
  mixins: [BasicPage],
  data() {
    return {
      tips: "",
      flowName: "",
      nodeName: "",
      compRate: 0
    };
  },
  methods: {
    onNavigated() {
      if (this.context && this.context.REQ_BODY && this.context.REQ_BODY.TASKID && this.context.REQ_BODY.TRADESTATUS=="1") {
        this.tips = "欢迎使用东莞农商远程银行系统，坐席正在为您服务。";
        this.flowName = this.context.REQ_BODY.FLOWNAME;
        this.nodeName = this.context.REQ_BODY.NODENAME;
        this.compRate = Math.round(this.context.REQ_BODY.COMPRATE);
      }
    },
  },
};
</script>
