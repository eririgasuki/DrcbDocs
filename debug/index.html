# 报文模拟

与P端确定好报文后，在交易页面父组件 PageContainerNavigator 中模拟接收报文。

在 workspace\main-stm\src\activities\PageContainerNavigator.vue 中找到生命周期 onInitialize：

async onInitialize() {
  // 获取session
  let result = Ameba.session;
  this.session = result;
  //注册remote事件
  RemoteCommandManager.addHandler(
    RemoteCommandTypes.openPage,
    this.onOpenPage.bind(this)
  );
  //初始化页面到视频请求
  const PagePath = "main-stm/src/modules/remote-floder/VideoConnect.vue";
  this.currentPage = await this.getPage(PagePath);
  console.log("currentpage %s", PagePath);
},

# 开发环境模拟调试

在初步编写完自己的交易页面后，可以直接在电脑上进行调试。

  1. 第一次调试需要找V端申请一个虚拟柜员号用于登陆应用。

  2. 在 workspace\main-stm 路径下运行 yarn dev(热更新开发模式)。

  1. 编译完毕后，打开位于 workspace\client\dotnet\Awp.SS.App.exe C端应用程序。

  1. 启动应用后点击视频服务即可跳转到开发的页面。
