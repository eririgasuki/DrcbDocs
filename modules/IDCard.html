# 注意点&可能的扩展改造

目前身份证模块读取的证件类型(IDTYPE)

IDTypeMap = {
  "02": "二代居民身份证",
  "18": "外国人永久居留身份证",
  "22": "港澳台居民居住证"
}

不同交易中，推送的证件类型字段不同（根据BUSISTEP取不同字段）

IDTypeFieldMap = {
  ReadCustIdNo_1: "IDTYPE",
  ReadLegalIdNo_1: "IDTYPE",
  ReadHandlerIdNo_1: "IDTYPE",
  ReadAgentIdNo_1: "AGENTIDTYPE",
  ReadPayerIdNo_1: "AGENTIDTYPE",
  ReadGuardIdNo_1: "GUARDIDTYPE"
};
IDType = this.find(IDTypeFieldMap[this.BUSISTEP]);
agentFlag = this.find("AGENTFLAG");

非本人办理时，页面提示语不同(AgentFlag ：0-本人,1-非本人) 最终提示语可能是：请插入代理人的港澳台居民居住证

agentMap = {
  ReadCustIdNo_1: "账户本人",
  ReadAgentIdNo_1: "代理人",
  ReadGuardIdNo_1: "监护人"
};
whose = agentFlag === "0" ? "您的" : agentMap[this.BUSISTEP] || "您的";
title = `请插入${whose + IDTypeMap[IDType]}`;

提交代理人、监护人证件信息的字段，需要前缀 AGENT 或 GUARD

rawInfo = {
  IDType,
  IDNO: IDData.ID,
  IDNAME: IDData.Name,
  IDADDRESS: IDData.Address,
  IDBIRTHDATE: IDData.Address,
  ISSUEOFFICE: IDData.Birthday,
  IDSEX: IDData.Sex === "女" ? "2" : "1",
  IDCERTDUEDATE: IDData.EndDate === "长期" ? "20991231" : IDData.EndDate
};
infoMap = {
  ReadCustIdNo_1: "",
  ReadLegalIdNo_1: "",
  ReadHandlerIdNo_1: "",
  ReadAgentIdNo_1: "AGENT",
  ReadPayerIdNo_1: "AGENT",
  ReadGuardIdNo_1: "GUARD"
};
info = {};
Object.keys(rawInfo).map(val => {
  info[infoMap[this.BUSISTEP] + val] = rawInfo[val];
});
this.submitAsync(info);

所有身份证都能读到 ID ，港澳台居民居住证能读取到特殊字段 PassNO 插入港澳台居民居住证时，IDNO字段送PassNo值，IDPASSNO字段送ID

this.submitAsync({
  IDNO: IDData.PassNO ? IDData.PassNO : IDData.ID,
  IDPASSNO: IDData.PassNO ? IDData.ID : ""
});

# 外设方法注意点

身份证页面引用的SDK不是新版的IDC模块，不能用IDC的方法，同方法的返回结果也会不同

import from { IDCardReader } from 'awp-plugin-device-idcard-reader'
// import from { IDCardReader } from 'awp-plugin-device-idc';

目前没有检查身份证状态和异常复位。如有需求，获取状态返回值中 Device 为0是正常

result = await IDCardReader.getStatusAsync()
result.Device != 0 && await IDCardReader.resetAsync()

不建议用cancelAsync，可能会导致应用的卡顿

// IDCardReader.cancelAsync()

不建议用Promise.then，回调函数套娃难以维护，建议async await

IDCardReader.waitCardTakenAsync()
  .then(XXX).catch(YYY)

try{
  await IDCardReader.waitCardTakenAsync();
  XXX
}catch(error){
  YYY
}
