# 注意点&可能的扩展改造

只有BUSISTEP === ReadPerUnitBankBook_2才有当前坐席选择的对公、对私字段 CSTMARK (1-对私 2-对公)。

读取存折只有track2和track3，与能直接读取到cardNumber的银行卡不同。

result = await PassbookPrinter.readTracksAsync();
track = result.track2 || result.track3
if (track.substring(0, 6) === "940062") {//对私存折
  vouNo = track.substring(40, 53); //凭证号
  accNo = track.substring(21, 36); //存折号
} else if (track.substring(14, 22) === "01561560") {//对公存折
  vouNo = track.substring(1, 14); //凭证号
  accNo = track.substring(22, 40); //存折号
}

# 外设方法注意点

因为cancelAsync可能会导致卡顿，所以未插入介质时关闭页面后，为了避免插入介质后函数继续执行，会使用return出去。

async onClose() {
  // PassbookPrinter.cancelAsync();
  !this.mediaInserted && (this.needReturn = true);
},

async onNavigated() {
  await PassbookPrinter.readTracksAsync(); //未插入介质关闭页面时，读取方法仍然在等待中
  if (this.needReturn) {
    return;
  }
  this.mediaInserted = true;
}

目前没有检查存折打印机状态和异常复位。如有需求，获取状态返回值中 Device 为0是正常

result = await PassbookPrinter.getStatusAsync()
result.Device != 0 && await PassbookPrinter.resetAsync()
